# MercApp - Frontend (Unidad 3)

SPA con Vue 3 (Composition API, SFC) que consume el API REST propio.

## Requisitos
- Node.js 18+

## Configuraci√≥n
1. Crear archivo `.env` (opcional). Por defecto ya apunta a `http://localhost:3000/api`:
   ```
   VITE_API_BASE=http://localhost:3000/api
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Ejecutar modo desarrollo:
   ```bash
   npm run dev
   ```

Abre `http://localhost:5173` en tu navegador.

MercApp - Unidad 3 (Aplicaciones Web)

Este proyecto corresponde a la Unidad 3 de la materia Aplicaciones Web.
Incluye un backend en Node/Express con MongoDB y un frontend en Vue 3 con Vite.

-------------------------
TECNOLOGÍAS UTILIZADAS
-------------------------
Backend:
- Node.js
- Express
- MongoDB con Mongoose
- Dotenv
- Cors
- Nodemon

Frontend:
- Vue 3
- Vite
- Vue Router
- Pinia

-------------------------
FUNCIONALIDADES
-------------------------
- Listado de productos
- Filtro por categorías
- Buscador por nombre
- Carrito de compras (Pinia)
- Vista de detalle de producto
- Edición de producto mediante formulario
- API REST completa (GET, POST, PUT, DELETE)
- Seed para insertar datos iniciales

-------------------------
SEED DE PRODUCTOS
-------------------------
Para ejecutar el seed:
1. Abrir la carpeta backend en terminal (click derecho → "Abrir en terminal integrado")
2. Ejecutar: npm run seed

-------------------------
CÓMO EJECUTAR EL PROYECTO
-------------------------
Backend:
1. Abrir carpeta backend en terminal
2. npm install
3. npm run dev
Disponible en: http://localhost:3000

Frontend:
1. Abrir carpeta frontend en terminal
2. npm install
3. npm run dev
Disponible en: http://localhost:5173
